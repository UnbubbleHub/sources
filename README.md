# Unbubble

Open tools for research and applications against social polarization.

Unbubble provides algorithms and APIs to surface diverse, high-quality perspectives on any news event or claim, helping users break out of information bubbles.

## Project structure

- `planning/` — Project plan and roadmap (EN / IT)
- `main.py` — Entry point
- `pyproject.toml` — Python project configuration

## Goals

- Given a news event, retrieve articles covering it from multiple perspectives
- Extract AI-generated metadata to measure perspective diversity and source quality
- Return a ranked list of articles that maximizes viewpoint coverage while prioritizing reliable sources
